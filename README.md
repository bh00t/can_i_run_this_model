# Can I Run This Model?

> Find which local AI models your PC can actually run — instant compatibility grades, tok/s estimates, and hardware-aware filtering.

**v1.4 — May 2026 · Single HTML file · No install · No backend**

🔗Live site →  **[can_i_run_this_model](https://bh00t.github.io/can_i_run_this_model/)**

---

## What is it?

Enter your RAM and VRAM (or hit Auto-Detect), and the page instantly scores every model in the catalog against your hardware — telling you exactly what will run, how fast, and on which settings. No guesswork, no vague "recommended specs."

---

## What's inside

- **51 open-source models** spanning Entry through Pro hardware tiers
- **S / A / B / C / F compatibility grades** based on your actual RAM + VRAM
- **Real-time tok/s estimates** tailored to your GPU
- **85 GPUs supported** — NVIDIA RTX 20/30/40/50, AMD RX 6000/7000, Intel Arc, Apple M1–M4, Qualcomm Snapdragon X
- **Auto-detect hardware** — one click to identify your GPU and memory
- **Filter by use case** — chat, code, reasoning, vision, RAG, tools
- **Model detail drawer** — full specs, quantization options, and Ollama install command
- **Shareable links** — share your hardware context via URL
- **Mobile friendly** — works on any screen size
